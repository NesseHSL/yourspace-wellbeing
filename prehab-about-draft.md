# PREHAB: STABILITY & STRENGTH FOR INJURY PREVENTION

## Who this is for

Whether you're returning from an injury, getting back into exercise after a break, or just starting out and wanting to do it properly — this programme is for you. Prehab is the foundational work that physios and movement specialists swear by: stability, strength, and body mechanics, built from the ground up.

## How the programme works

Six sessions, added progressively over 28 days — each one stacking onto the last so your body builds real, lasting strength without burning out.

| Days | What you're doing | Approx. time |
|---|---|---|
| Days 1–2 | Ground Up (feet & ankles) | ~15 min |
| Day 3 | Rest | |
| Days 4–6 | Ground Up + Knees, Quads & Hamstrings | ~35 min |
| Day 7 | Rest | |
| Days 8–9 | Ground Up + Glutes & Hips | ~35 min |
| Day 10 | Rest | |
| Days 11–12 | Ground Up + Knees + Glutes & Hips | ~55 min |
| Days 13–14 | Rest | |
| Days 15–16 | Ground Up + Core | ~30 min |
| Day 17 | Rest | |
| Days 18–19 | Core + Stand Tall (upper body) | ~35–40 min |
| Day 20 | Rest | |
| Days 21–24 | The Full Circuit | ~50 min |
| Days 25–28 | Rest & stretch only | |

Every session has beginner, intermediate, and advanced options, so you can always work at the right level for your body.

## Work at your own pace

You have 90 days of access — so there's no rush. If a session needs more time, stay with it. Repeat it. Your body will tell you when it's ready to move on.
